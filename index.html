Що вивчає школа Logika?

PYTHON

Для чого використовується python? - Puthon використовується для розробки вебсайтів, додатків та ігор, аналізу даних, системного адміністрування, тестування.

JAVASCRIPT

Для чого використовується javascript? - Java script використовується для запису послідовних операцій — «сценаріїв» чи «скриптів».

HTML

Для чого використовується html? - Мова html придумана для перегляду веб-сторінок в бравзері та створення сайтів за використанням тегів.

CSS

Для чого використовується CSS? - Програмувальна мова CSS використовується в для опису зовнішнього вигляду, та скорочення часу на розробітку сайту.

SCRATCH

Для чого використовується SCRATCH? - Програмувальна мова CSS використовується в для опису зовнішнього вигляду, та скорочення часу на розробітку сайту.
